# Yondu Inc. — Full-Stack Delivery

## Context

A year at Yondu Inc. as a Software Engineer, delivering full-stack work against tight deadlines. The role emphasized consistent execution, test discipline, and source-control hygiene as much as the code itself.

## Scope of work

- Translated technical designs into functional code across backend and frontend.
- Implemented efficient program logic and data manipulation.
- Developed and maintained unit tests to lock in behavior.
- Diagnosed and resolved software issues promptly as they arose.
- Managed project source code repositories.

## Outcome

- Maintained a 100% on-time project delivery record.
- Raised code coverage by ~60% through a deliberate unit-testing strategy.

## Stack notes

- **ASP.NET Core + C#** on the backend
- **Angular + TypeScript** on the frontend
- **SQL Server** for data
- **Git** for source control workflows
