# Sense Software Solutions — Aviation & Logistics

## Context

A six-month specialized role at Sense Software Solutions, focused on aviation and logistics applications delivered in collaboration with business architects in Singapore. The work spanned service-layer design (WCF + Web API) and an Angular front end, under agile rituals that crossed time zones.

## Scope of work

- Designed and developed web applications using ASP.NET for the aviation and logistics domain.
- Created WCF web services and Web APIs to support the application layer.
- Developed Angular web applications on the client side.
- Collaborated with business architects in Singapore to coordinate requirements across teams and time zones.

## Outcome

- Delivered an aviation logistics application adopted by multiple airlines.
- Implemented robust WCF services handling high-volume transactional workloads.

## Stack notes

- **ASP.NET + C#** on the backend
- **WCF + Web API** as the service layer
- **Angular** on the client
- **SQL Server** for the system of record
