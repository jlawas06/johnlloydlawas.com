# Lear Corporation — Software Developer Intern

## Context

My first professional role. Eleven months at Lear Corporation as a Software Developer Intern, working on in-house enterprise web applications. This is where I learned how enterprise development actually runs day to day — process, reviews, handoffs, maintenance — alongside the core stack of ASP.NET MVC and SQL Server.

## Scope of work

- Designed and developed in-house web applications using ASP.NET MVC.
- Learned enterprise development practices and patterns on the job.
- Participated in code reviews and team development processes.
- Assisted in maintaining existing applications.

## Outcome

The internship converted into a full-time developer track on the strength of shipped work. Along the way I contributed internal tools that measurably improved day-to-day team productivity.

## Stack notes

- **ASP.NET MVC + C#** for server-rendered web applications
- **SQL Server** as the system of record
- **JavaScript, HTML, CSS** on the client side
